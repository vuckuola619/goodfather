# Pre-Implementation Hook
# Fires BEFORE any code is written — enforces Superpowers + Karpathy discipline

## Checklist (MUST complete before writing code)

### 1. Design Read (Taste Skill)
- [ ] State the design read: "Reading this as: <page kind> for <audience> with <vibe>"
- [ ] Set dial values: VARIANCE / MOTION / DENSITY

### 2. Assumption Surface (Agent Skills)
- [ ] List all assumptions being made
- [ ] Identify any ambiguity in the request
- [ ] If confused, STOP and ask ONE clarifying question

### 3. Simplicity Check (Karpathy)
- [ ] Is this the minimum code that solves the problem?
- [ ] Am I adding anything that wasn't asked for?
- [ ] Could a senior engineer simplify this?

### 4. Existing Code Awareness (Karpathy)
- [ ] Read ALL related files before proposing changes
- [ ] Check if similar functionality already exists
- [ ] Match existing style and patterns

### 5. Token Efficiency (RTK/Graphify)
- [ ] Am I loading only the context I need?
- [ ] Can I reference files instead of re-reading them?
- [ ] Am I using the most efficient output format?
